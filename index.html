发表于2023-07-01更新于2023-07-01

hexo安装

通过gitee创建博客仓库

  1. 注意：仓库名称和gitee用户名称一致

  2. 生成/添加 SSH 公钥

  3. 查看邮箱，账户

    1. git config –global user.name
      git config –global user.email

  4. 配置邮箱，账户

    1. git config –global user.email abc@qq.com # 设置邮箱
      git config –global user.name ‘abc’ # 设置用户名

  5. 本地生成ssh公钥

    1. ssh-keygen -t rsa -C “xxxxx@xxxxx.com“

  6. gitee创建公钥

  7. 测试连接

    1. ssh -T git@gitee.com

      显示如下内容

      表示连接成功

      Hi “您的用户名”! You’ve successfully authenticated, but GitHub does not provide shell access.

hexo建站，主题配置

  1. 安装hexo

    1. npm install -g hexo # 通过npm安装hexo

  2. 初始化hexo

    1. hexo init blog # 初始化创建，会再桌面创建blog文件夹
      cd blog # 进入blog目录
      npm install # 进一步安装hexo所需文件

    2. 初始化目录情况

  3. 启动hexo

    1. hexo clean # 清除所有记录 /hexo c
      hexo generate # 生成静态网页 /hexo g
      hexo server # 启动服务 /hexo s

  4. 配置连接 Gitee

  5. 部署博客

    1. npm install hexo-deployer-git –save

  6. Gitee Pages 设置
